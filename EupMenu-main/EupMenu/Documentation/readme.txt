---------------
EUP MENU README
---------------
Requirements:
1) RAGE Plugin Hook 0.37+
2) RAGENativeUI 1.8.1+
3) EUP Law & Order 8.3
4) (Optional but recommended) EUP Serve & Rescue 1.5

Installation:
1) Place RAGENativeUI.dll into your root GTA V folder (where GTA5.exe is).
2) Extract the contents of the Plugins folder in the archive into your GTA V Plugins folder.
3) Open the RPH console in-game (default F4) and type "LoadPlugin EUPMenu.dll" (without quotations).
You can set the menu to auto-load on startup in your RAGE Plugin Hook settings.
